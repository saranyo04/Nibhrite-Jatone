# Nibhrite Jatone

Website for Nibhrite Jatone Homestay in Santiniketan.

Production URL: https://nibhritejatone.in

## Tech Stack

- Next.js
- React

## Project Structure

```text
src/
  app/
    gallery/
  components/
    sections/
    ui-custom/
  data/
  lib/
public/
  images/
    experience/
    gallery/
      exterior/
      food/
      interior/
      rooms/
```

## Development Commands

```bash
npm install
npm run dev
npm run lint
npx tsc --noEmit
npm run build
```

All content lives in `src/data/site-data.ts`.
